## üìÑ Overview

This project is a static website built with **HTML5** and styled using **Tailwind CSS**. It serves as a digital resume and portfolio for **Kingshuk Datta**, a Computer Science & Engineering student.

The site features a clean, modern design with smooth scrolling navigation and custom animations.

## ‚ú® Key Features

* **Responsive Design:** Fully adaptive layout that works on mobile, tablet, and desktop (powered by Tailwind Grid and Flexbox).
* **Interactive Navigation:**
    * Fixed top navigation bar.
    * **Scroll Spy:** The navigation links automatically highlight based on the section currently visible on the screen.
    * **Smooth Scrolling:** Clicking a link glides to the section rather than jumping abruptly.
* **Animations:** Custom CSS keyframe animations (`fade-in-down` and `fade-in-up`) on the hero section.
* **Dynamic Footer:** Automatically updates the copyright year using JavaScript.
